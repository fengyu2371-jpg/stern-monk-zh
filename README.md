# stern-monk-zh-tw v24｜地點編輯版

這是修士 Bot 的 Railway 部署精簡版。

## 部署檔案

```text
main.py
requirements.txt
railway.toml
data/
  faq_zh_tw.json
  tutorials_zh_tw.json
  dialogue.json
```

程式碼只保留 `main.py` 一份；`data/` 是教學與台詞資料。

## 指令

```text
/學生資料
/修士狀態
```

原本的 `/建立修士面板` 已移除。

## 主要規則

- `/學生資料` 會直接公開顯示使用者自己的學生資料。
- 其他玩家看得到，但不能操作按鈕。
- 其他玩家點擊時只收到私密提示，不會改動別人的資料。
- 本人每次有效操作都會重新計時。
- 5 分鐘沒有操作後，按鈕與選單會被移除，公開資料仍保留。
- 要再次操作，重新輸入 `/學生資料`。
- 我的地點採總覽＋下拉選單管理，地點可刪除，刪除前有二次確認。
- 商店街會讀取所有公開店鋪，並顯示店主／經營者。


## v24 地點編輯

在「我的地點」選擇單一地點後，可按：

```text
✏️ 編輯地點
```

可修改：

- 地點名稱
- 店主／經營者／居住者
- 所在區域
- 目前狀態
- 地點簡介

地點類型、來源與公開狀態不放在編輯表單中：
公開狀態仍使用原本的「公開顯示」按鈕調整。
其他玩家仍只能查看，不能編輯別人的地點。
